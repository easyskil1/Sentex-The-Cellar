# Sentex: The Cellar

Top-down roguelike, built on a **custom Canvas engine + TypeScript + Vite**, with
no game-engine dependency. All graphics are **procedural** (drawn from code, no asset files).

## Play

Live build: **https://easyskil1.github.io/Sentex-The-Cellar/**

## Controls

| Key | Action |
|---|---|
| `W A S D` | move |
| `↑ ↓ ← →` | shoot |
| 🖱 mouse | aimed shooting |
| `E` | active skill |
| `B` | drop bomb |
| `P` / `Esc` | pause |
| `M` | mute |

## Development

```bash
npm install
npm run dev        # dev server at http://localhost:5173
npm run build      # type-check + production build → dist/
npm run preview    # serve the production build
```
